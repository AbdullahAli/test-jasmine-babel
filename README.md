# test-jasmine-babel

request some json, process it, count some stuff and write it to a file using es6 and promises.
## Setup

```
npm install
```

## Test

Before you should have global installation of `jasmine` package.

```
npm test
```

## Support 

Current support version:

 - `Babel ^6.3.17`

Previously support:

 - `Babel ^5.8.29`
